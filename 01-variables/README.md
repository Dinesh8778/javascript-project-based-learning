# Lesson 01 – Variables

## Overview

This lesson introduces variables in JavaScript and explains how to store, update, and use data in programs.

## Concepts Covered

- Variables
- let
- const
- var
- Variable declaration
- Variable reassignment
- Naming conventions
- camelCase
- console.log()

---

## Files

| File | Description |
|------|-------------|
| basic_variables.js | Basic variable declaration and reassignment |
| product_inventory.js | Product information exercise |
| student_profile.js | Mini project using variables |

---

## Exercises

### Exercise 1

- Declare variables
- Print values
- Update values

### Exercise 2

Create a product inventory using variables.

### Mini Project

Create a Student Profile application that stores and prints:

- Name
- Age
- Department
- College
- CGPA

Then update the CGPA and print it again.
