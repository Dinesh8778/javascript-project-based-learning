# Lesson 02 – Data Types

## Overview

This lesson introduces JavaScript data types and explains how different kinds of values are stored in variables.

Understanding data types is essential because JavaScript treats numbers, strings, booleans, and other values differently during program execution.

---

## Concepts Covered

- Primitive Data Types
- Number
- String
- Boolean
- Undefined
- Null
- BigInt
- Symbol
- Object (Introduction)
- typeof Operator

## Learning Outcomes

After completing this lesson, I can:

- Identify JavaScript data types
- Choose the correct data type for different kinds of data
- Use the `typeof` operator
- Understand the difference between `null` and `undefined`
- Store and display values correctly

---

## Common Mistakes

- Confusing `"20"` with `20`
- Using `"true"` instead of `true`
- Mixing `null` and `undefined`
- Storing phone numbers as numbers instead of strings
