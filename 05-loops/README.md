# Lesson 05 – Loops

## Overview

This lesson introduces loops in JavaScript, allowing repetitive tasks to be performed efficiently. It covers different looping statements, loop control keywords, and nested loops through practical exercises and a mini project.

---

## Concepts Covered

- `for` Loop
- `while` Loop
- `do...while` Loop
- Nested Loops
- `break`
- `continue`
- Infinite Loops

## Practice Exercise

Create a JavaScript program to complete the following tasks.

### Exercise 1

Using a `for` loop, print numbers from **1 to 10**.

---

### Exercise 2

Using a `while` loop, print numbers from **10 to 1**.

---

### Exercise 3

Using a `do...while` loop, print numbers from **1 to 5**.

---

### Exercise 4

Using a `for` loop, print all **even numbers** from **1 to 20**.

---

### Exercise 5

Using a `for` loop, print all **odd numbers** from **1 to 20**.

---

### Exercise 6

Print the multiplication table of **7**.

Example:

```text
7 × 1 = 7
7 × 2 = 14
...
7 × 10 = 70
```

---

### Exercise 7

Print numbers from **1 to 20**.

Skip **10** using `continue`.

---

### Exercise 8

Print numbers from **1 to 20**.

Stop the loop when the number reaches **15** using `break`.

---

## Common Mistakes

- Forgetting to update the loop variable.
- Creating infinite loops.
- Using the wrong loop type for the problem.
- Forgetting that `do...while` executes at least once.
- Missing `break` or incorrectly using `continue`.
